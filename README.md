# Expense Tracker Project

A beginner full-stack expense tracker project using:

- HTML
- CSS
- JavaScript
- Python Flask
- OOP
- File Handling
- C Language

## Features

- Add Expense
- View Expenses
- Store Data in File
- Expense Report Generator

## Run Backend

```bash
pip install -r requirements.txt
python app.py
```

## Open Frontend

Open index.html in browser.
